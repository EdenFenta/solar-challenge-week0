# Notebooks Folder

This folder contains all Jupyter notebooks used for EDA (Exploratory Data Analysis) of solar datasets.

## Contents

- `benin_eda.ipynb` → EDA for Benin dataset
- `sierra_leone_eda.ipynb` → EDA for Sierra Leone dataset
- `togo_eda.ipynb` → EDA for Togo dataset

## Purpose

- Perform profiling, cleaning, and statistical analysis.
- Create visualizations like scatter plots, bubble charts, and wind rose.
- Export cleaned datasets to `data/` folder.

## How to Run

1. Activate the virtual environment.
2. Install dependencies (`pip install -r requirements.txt`).
3. Open the notebook in Jupyter or VSCode.
4. Run each cell sequentially.
